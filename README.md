# CalmSense

CalmSense is an AI-powered mindfulness and mental well-being platform designed to help users manage stress and achieve tranquility. By leveraging machine learning models to analyze user mood and physiological inputs, CalmSense dynamically generates personalized relaxation exercises, customized meditation guides, and soothing audio clips tailored to each individual's state of mind. The application consists of a modern Next.js frontend, a robust Python-based backend handling machine learning analysis and generation, and a curated library of calming audio assets.
